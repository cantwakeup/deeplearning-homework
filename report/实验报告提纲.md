## 第一章 NumPy 实现 MLP

### 原题目

编写 MLP，只能使用 NumPy 编写，不能直接使用 PyTorch 中的 linear、sigmoid 等函数；完成输入层、隐含层和输出层；编写损失函数、激活函数、训练过程和预测过程；回归或分类均可，数据集可自行选择。

### 题目分析及解题思路

选用三分类螺旋数据集。该数据集不是线性可分的，可以检验隐含层和非线性激活函数是否有效。模型实现包括权重初始化、ReLU/Tanh/Sigmoid、Softmax、交叉熵损失、L2 正则、普通 GD、Momentum 和 Adam。

### 实验过程

基础实验使用 `01_numpy_mlp/run_experiment.py`。扩展实验使用 `01_numpy_mlp/sweep_mlp.py`，比较：

- 激活函数：ReLU、Tanh、Sigmoid
- 隐藏层宽度：16、32、64、128
- 学习率：0.1、0.05、0.01、0.005
- 优化器：GD、Momentum、Adam
- Top 配置多随机种子复验：0、1、2

### 实验结果及分析

基础结果：训练 1500 epoch，测试准确率 0.9667。扩展实验最佳配置为 ReLU、hidden_dim=16、Adam、lr=0.01，多种子平均测试准确率 0.9917，标准差约 0.0068。结果说明，合理优化器和学习率能够显著提升手写 MLP 的收敛效果；多随机种子复验可避免偶然初始化带来的误判。

## 第二章 PyTorch CNN 手写数字识别

### 原题目

编写一个卷积网络，对手写体数字进行识别。数据集使用 torch 中自带的样本数据集；卷积网络结构、激活函数、损失函数和超参数自定义；保存训练结果；分析多种情况并保留分析结果。

### 题目分析及解题思路

使用 MNIST 作为 10 分类任务。CNN 通过局部连接和参数共享提取笔画、边缘等局部特征。模型由两组卷积块和全连接分类器组成，支持调整卷积核大小、通道数、BatchNorm、Dropout 和池化。

### 实验过程

基础实验使用 `02_mnist_cnn/train_cnn.py`。扩展实验分两部分：

- `02_mnist_cnn/sweep_cnn.py`：Adam、SGD、RMSprop 与多档学习率组合，共 9 组。
- `02_mnist_cnn/sweep_arch_cnn.py`：卷积核大小、通道宽度、BatchNorm、Dropout、池化消融。

### 实验结果及分析

基础 CNN 测试准确率 0.9900。优化器/学习率对比中，RMSprop + 0.0001 测试准确率 0.9896，Adam + 0.001 测试准确率 0.9894。架构消融中，5x5 卷积核、32/64 通道、BatchNorm、Dropout=0.25 的配置最好，测试准确率 0.9915。无池化配置降至 0.9845，说明适度下采样对短周期训练和泛化有帮助。

## 第三章 PyTorch 实现 ResNet

### 原题目

使用 PyTorch 实现 ResNet，层数不限，建议 50 层，数据集可以使用 MNIST，完成训练和预测。

### 题目分析及解题思路

基础实现保留任务书建议的 ResNet-50。扩展实现加入 ResNet-18、ResNet-34、Plain-18、MNIST stem 和 ImageNet stem 对比。ResNet-50 使用 Bottleneck 模块，ResNet-18/34 使用 BasicBlock。通过残差连接对比说明深层网络训练稳定性的来源。

### 实验过程

基础实验使用 `03_resnet_mnist/train_resnet.py`。扩展实验使用 `03_resnet_mnist/sweep_resnet.py`，比较：

- 深度：ResNet-18、ResNet-34、ResNet-50
- 残差：ResNet-18 与 Plain-18
- 输入 stem：MNIST 3x3 stem 与 ImageNet 7x7 stem
- 优化器和学习率：Adam、SGD、RMSprop 及不同学习率

### 实验结果及分析

ResNet-50 基线测试准确率 0.9872。扩展实验中 ResNet-34 最优，测试准确率 0.9931；ResNet-18 为 0.9917；Plain-18 为 0.9781，明显低于带残差连接的 ResNet-18，说明残差连接对稳定训练有实际贡献。ImageNet stem 为 0.9885，低于 MNIST stem，说明小尺寸输入不适合过早强下采样。
